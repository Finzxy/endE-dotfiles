# endE-dotfiles
fast, lightweight, bloatless. (work in progress)

## v0.01
### how to use
```
git clone https://github.com/finzxy/endE-dotfiles.git
mv ~/endE-dotfiles/Config/.config/quickshell ~/.config/

# optional :
mv ~/endE-dotfiles/Config/.config/hypr ~/.config/
mv ~/endE-dotfiles/Config/.config/matugen ~/.config/
```
